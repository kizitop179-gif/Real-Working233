# 11v11 Football Game - Web Version

## 🎮 Play Online!

You can play this game directly in your web browser - no installation needed!

### 🔗 Direct Links:
- **GitHub Pages:** https://kizitop179-gif.github.io/Real-Working233/web/
- **Local Play:** Open `index.html` in your browser

## How to Play

### Basic Controls
1. **Click on a player** to select them (shown with yellow outline)
2. **Use keyboard controls** to move the selected player
3. **Pass/Shoot** to move the ball toward the goal
4. **Score goals** by getting the ball into the opponent's goal area

### Team A (Blue) - Left Side
- **W/S** - Move Up/Down
- **A/D** - Move Left/Right  
- **Space** - Pass/Shoot

### Team B (Red) - Right Side
- **↑/↓** - Move Up/Down
- **←/→** - Move Left/Right
- **Enter** - Pass/Shoot

### General
- **Click** - Select a player
- **R** - Reset game and scores

## Game Features

✅ **11 Players Per Team** (4-3-3 Formation)
- 1 Goalkeeper
- 4 Defenders
- 3 Midfielders
- 3 Forwards

✅ **Full Game Mechanics**
- Real-time score tracking
- Game timer (MM:SS format)
- Physics-based ball and player movement
- Collision detection
- Goal detection

✅ **Web-Based**
- No installation required
- Play directly in browser
- Works on desktop and mobile
- Responsive design

## Game Rules

- Score by getting the ball into the opponent's goal area
- Each team has 11 players in 4-3-3 formation
- Players can move freely across the pitch
- Ball bounces off walls and players
- Game timer tracks elapsed time
- Pass between players to control the ball

## Running Locally

### Option 1: Open Directly
```bash
# Simply open index.html in your browser
open index.html
```

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000
# Then visit: http://localhost:8000/web/

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js
npx http-server
# Then visit: http://localhost:8080/web/
```

## Game Tips

💡 **Strategy Tips:**
- Click on defensive players to clear the ball
- Use forwards (strikers) to attack and score
- Pass between midfielders for better positioning
- Control the ball near the goal to score
- Move your goalkeeper to block the goal

🎯 **Winning Tips:**
- Don't leave your goal undefended
- Pass quickly to maintain possession
- Position forwards strategically
- Use the wings to create attacking opportunities
